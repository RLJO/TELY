Pay Employee Expense In Payroll

"Reimburse by Payslip"

This module allows paying/reimbursing employee expenses in the Payroll.

Description

The expense reimbursement process allows employers to pay back employees who have spent their own money for business-related expenses.

The “Pay Employee Expense In Payroll” integrates the Expense application with the Payroll application to reflect expense reimbursement in the employee’s payslip. In other words, this module allows deferring the expense amount for including in the next payslip instead of paying back to the employee immediately.
